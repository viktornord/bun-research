# React with Bun / Node

## Installation

```
npm install | bun install
```

<br/>

## Available Scripts

### Develop
```
npm start | npm run start | bun start | bun run start | bun dev
```
### Test
```
npm test | npm run test | bun test | bun run test
```
### Build
```
npm run build | bun run build
```
### Eject
```
npm run eject | bun run eject
```
